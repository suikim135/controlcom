Windows용 ControlCom 설치

폰 앱이 PC 명령을 전달하려면, PC에 ControlCom Agent가 필요합니다. Spotify·Discord처럼 PC용 동반 프로그램을 한 번만 설치하면 됩니다.

Windows용 ControlCom 설치

Windows 10 / 11 · 다운로드 후 설치 마법사 안내를 따라 진행하세요.

설치 후 이렇게 진행됩니다
  1. 설치 마법사에서 다음 → 설치 → 완료
  2. PC에 ControlCom Agent가 실행됩니다 (백그라운드)
  3. 작업 표시줄 숨겨진 아이콘(^)에서 ControlCom 아이콘 확인
  4. 우클릭 → 페어링 QR 열기 → 폰 앱에서 스캔

이 프로그램이 하는 일

  • 같은 Wi-Fi에 있는 폰의 요청만 받습니다
  • 절전, 음소거, 모니터 전환, 저장 후 종료 명령을 PC에서 실행합니다
  • 인터넷 밖에서 PC를 조종하지 않습니다

보안 안내

  • 같은 Wi-Fi에서만 동작합니다
  • 외부 서버로 개인 데이터를 보내지 않습니다
  • 페어링한 폰만 PC를 제어할 수 있습니다
  • 공유기 포트포워딩은 하지 마세요

제거 방법

Windows 설정 → 앱 → 설치된 앱에서 ControlCom Agent를 찾아 제거할 수 있습니다.
