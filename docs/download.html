ControlCom Agent

Windows PC에서 ControlCom Android 앱의 명령을 받는 공식 PC 프로그램입니다.

Windows 설치 파일 다운로드 (Setup.exe)

최신 버전: GitHub Releases

설치 후: 로그인 시 자동 실행(선택) → 브라우저에 페어링 QR 표시 → 폰 앱에서 QR 스캔

보안 안내

  • 같은 Wi-Fi(LAN)에서만 동작합니다. 인터넷 원격 제어는 지원하지 않습니다.
  • 외부 서버로 사용자 데이터를 보내지 않습니다.
  • 페어링한 기기만 PC를 제어할 수 있습니다.
  • 공유기 포트포워딩(7847)은 하지 마세요.

요구 사항

  • Windows 10 / 11
  • PC와 폰이 같은 Wi-Fi에 연결
