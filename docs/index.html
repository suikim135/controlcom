ControlCom

같은 Wi-Fi에서 PC를 조종하는 Android 리모컨

절전, 음소거, 모니터 전환, 저장 후 종료 — PC 앞에 가지 않고 폰 버튼 몇 번으로 끝냅니다.

Google Play에서 받기 Windows PC 프로그램

처음 쓰는 방법
  1. 폰에 ControlCom 앱 설치 (Play Store)
  2. PC에 ControlCom Agent 설치
  3. PC 화면의 QR 코드를 폰 앱에서 스캔

기능

  • PC 절전 (Sleep)
  • 시스템 음소거 토글
  • 주 모니터만 / 듀얼 모니터 복구
  • 저장 후 종료

보안

  • 같은 Wi-Fi(LAN)에서만 동작 — 인터넷 원격 제어 없음
  • 외부 서버로 사용자 데이터를 보내지 않음
  • 페어링한 기기만 PC를 제어할 수 있음
